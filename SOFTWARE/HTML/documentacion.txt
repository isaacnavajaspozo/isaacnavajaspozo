================================================
====================[ HTML ]====================
================================================

# body 
El elemento <body> contiene todo el contenido visible de una página web. Aquí se incluyen todos los elementos que se mostrarán al usuario, como textos, imágenes y enlaces.

# footer 
El elemento <footer> almacena el pie de página, que generalmente contiene información sobre el autor, derechos de autor, enlaces de contacto o un mapa del sitio.

# nav 
El elemento <nav> se utiliza para contener los enlaces de navegación, tanto internos (dentro del mismo sitio web) como externos (hacia otros sitios web).

# main 
El elemento <main> representa la parte principal del contenido del documento, que debe ser único y relevante para la página actual.

# article 
El elemento <article> se usa para encapsular artículos, entradas de blog o contenido independiente que tiene sentido por sí mismo.

# aside 
El elemento <aside> contiene información secundaria que puede complementar el contenido principal, como una biografía del autor o anuncios publicitarios.

# section 
El elemento <section> agrupa secciones dentro de un documento, facilitando la organización del contenido en temas o categorías.

# hgroup 
El elemento <hgroup> agrupa títulos secundarios (h1-h6) para crear una jerarquía clara y estructurada de encabezados.

# figure y figcaption 
Los elementos <figure> y <figcaption> se utilizan juntos para agrupar imágenes (o ilustraciones) y sus descripciones o leyendas.

# div 
El elemento <div> crea divisiones o secciones en un documento, lo que permite agrupar y organizar el contenido de manera efectiva.

=====================================================
====================[ Etiquetas ]====================
=====================================================

# h1-h6 
Los elementos <h1> a <h6> son títulos con diferentes niveles de jerarquía, donde <h1> es el más importante y <h6> el menos.

# p 
El elemento <p> se utiliza para definir párrafos, que son bloques de texto relacionados.

# cite 
El elemento <cite> se utiliza para indicar citas o títulos de obras.

# blockquote 
El elemento <blockquote> se utiliza para mostrar citas en bloque, generalmente con sangrías.

# hr 
El elemento <hr> crea un separador horizontal, que se utiliza para dividir secciones de contenido.

# br 
El elemento <br> inserta un salto de línea, permitiendo que el texto continúe en la siguiente línea.

# strong, b 
Los elementos <strong> y <b> se utilizan para mostrar texto en negrita, pero <strong> también implica que el texto es importante.

# i, em 
Los elementos <i> y <em> se utilizan para mostrar texto en cursiva, con <em> indicando énfasis.

# u 
El elemento <u> se utiliza para subrayar texto, pero su uso debe ser considerado en el contexto actual del diseño.

# small 
El elemento <small> se utiliza para mostrar texto pequeño, como aclaraciones o notas al pie.

# big 
El elemento <big> se utiliza para mostrar texto más grande, aunque su uso es menos común en diseño moderno.

# sub 
El elemento <sub> se utiliza para mostrar subíndices, como en fórmulas químicas.

# del 
El elemento <del> se utiliza para mostrar texto tachado, indicando que ha sido eliminado o cambiado.

# q 
El elemento <q> se utiliza para citas cortas, que normalmente se muestran con comillas.

# abbr 
El elemento <abbr> se utiliza para abreviaciones, ofreciendo una forma más corta de referirse a términos largos.

# dfn 
El elemento <dfn> se utiliza para definir términos, resaltando su primer uso.

# address 
El elemento <address> se utiliza para mostrar información de contacto, generalmente en el pie de página.

# span 
El elemento <span> se utiliza para aislar texto dentro de un bloque para aplicar estilos específicos.

# iframe 
El elemento <iframe> se utiliza para incrustar otro documento HTML dentro de la página actual.

# meta 
El elemento <meta> se utiliza para definir información útil sobre el documento, como descripciones y palabras clave para motores de búsqueda.

# source 
El elemento <source> especifica recursos de audio, video o imágenes, facilitando la selección del formato adecuado.

==================================================
====================[ Listas ]====================
==================================================

# ol 
El elemento <ol> se utiliza para crear una lista numerada, donde cada elemento se presenta en un orden específico.

# ul 
El elemento <ul> se utiliza para crear una lista no numerada, donde los elementos se presentan sin un orden particular.

# li 
El elemento <li> define un elemento de una lista, ya sea dentro de una lista ordenada (<ol>) o no ordenada (<ul>).

# dl 
El elemento <dl> se utiliza para crear una lista descriptiva, que incluye términos y sus definiciones.

# dt 
El elemento <dt> define un término dentro de una lista descriptiva.

# dd 
El elemento <dd> proporciona detalles o descripciones para un término definido en una lista descriptiva.

=================================================
====================[ Links ]====================
=================================================

# a 
El elemento <a> se utiliza para crear enlaces que permiten a los usuarios navegar a otras páginas o recursos.

# mailto 
El atributo mailto en un enlace permite abrir el cliente de correo electrónico del usuario con una dirección de correo predefinida.

# target 
El atributo target se utiliza para abrir un enlace en una nueva pestaña o ventana.

# download 
El atributo download en un enlace permite a los usuarios descargar un archivo en lugar de navegar hacia él.

====================================================
====================[ Imágenes ]====================
====================================================

# img 
El elemento <img> se utiliza para incrustar imágenes en una página web.

# src 
El atributo src define la ruta de la imagen que se va a mostrar.

# alt 
El atributo alt proporciona texto alternativo que describe la imagen, útil para accesibilidad y SEO.

# title 
El atributo title ofrece información adicional sobre la imagen, que aparece como un tooltip al pasar el cursor.

# figure y figcaption 
Los elementos <figure> y <figcaption> se utilizan juntos para agrupar imágenes y sus leyendas, mejorando la accesibilidad y comprensión.

==================================================
====================[ Tablas ]====================
==================================================

# table 
El elemento <table> se utiliza para crear una tabla que organiza datos en filas y columnas.

# tr 
El elemento <tr> define una fila dentro de una tabla.

# th 
El elemento <th> se utiliza para definir una celda de encabezado, que se destaca generalmente en negrita y centrado.

# td 
El elemento <td> define una celda dentro de una fila de una tabla.

# thead 
El elemento <thead> se utiliza para agrupar el encabezado de una tabla.

# tbody 
El elemento <tbody> se utiliza para agrupar el cuerpo de la tabla, que contiene los datos.

# tfoot 
El elemento <tfoot> se utiliza para agrupar el pie de la tabla, que a menudo contiene totales o resúmenes.

========================================================
====================[ Formularios ]=====================
========================================================

# form 
El elemento <form> crea un formulario para enviar datos al servidor, permitiendo la interacción del usuario.

    ## form - action: 
    El atributo action define la URL a la que se envían los datos del formulario.

    ## form - method: 
    El atributo method especifica el método HTTP (GET o POST) que se utilizará para enviar los datos.

    ## form - enctype multipart/form-data: 
    El atributo enctype permite subir archivos en el formulario, necesario para el manejo de archivos adjuntos.

# input 
El elemento <input> se utiliza para crear campos de formulario, que pueden ser de varios tipos como texto, contraseña, etc.

# textarea 
El elemento <textarea> crea un campo de texto de múltiples líneas para entradas más largas.

# radio 
El elemento <input type="radio"> se utiliza para crear botones de opción, permitiendo seleccionar solo una opción de un grupo.

# checkbox 
El elemento <input type="checkbox"> se utiliza para crear casillas de verificación, permitiendo seleccionar múltiples opciones.

# select 
El elemento <select> se utiliza para crear un menú desplegable donde el usuario puede seleccionar una opción.

# option 
El elemento <option> define una opción dentro de un menú desplegable (<select>).

# button 
El elemento <button> se utiliza para crear un botón que puede activar acciones cuando se hace clic.

# label 
El elemento <label> se utiliza para asociar etiquetas a campos de formulario, mejorando la accesibilidad.

# fieldset 
El elemento <fieldset> se utiliza para agrupar campos en formularios, proporcionando una mejor organización visual.

# legend 
El elemento <legend> define un título para un fieldset, describiendo el propósito del grupo de campos.

========================================================
========================[ CSS ]=========================
========================================================

# padding 
El padding define el espacio interno entre el contenido de un elemento y su borde, creando separación visual.
Ejemplo: padding: 10px 20px; /* superior e inferior, derecha e izquierda */

# margin 
El margin define el espacio externo alrededor de un elemento, separándolo de otros elementos en la página.
Ejemplo: margin: 15px; /* espacio uniforme alrededor del elemento */

# box-sizing 
La propiedad box-sizing define cómo se calcula el ancho y alto de un elemento, afectando el modelo de caja.
Ejemplo: box-sizing: border-box; /* incluye padding y border en el ancho y alto */

# overflow 
La propiedad overflow controla cómo se muestra el contenido que desborda el área de un elemento.
Ejemplo: overflow: hidden; /* oculta el contenido que desborda */

# background 
La propiedad background permite establecer el color de fondo, imagen de fondo, y otros atributos en un solo lugar.
Ejemplo: background: #f0f0f0 url('imagen.jpg') no-repeat center; /* color de fondo, imagen, no repetir, centrado */

# opacity 
La propiedad opacity controla la opacidad de un elemento, donde 0 es completamente transparente y 1 es completamente opaco.
Ejemplo: opacity: 0.5; /* 50% de opacidad */

# display 
La propiedad display define cómo se muestra un elemento en la página, con valores como block, inline y none.
Ejemplo: display: flex; /* utiliza flexbox para el diseño */

# float 
La propiedad float permite que un elemento se desplace a la izquierda o derecha, afectando el flujo de otros elementos.
Ejemplo: float: left; /* flota el elemento a la izquierda */

# clear 
La propiedad clear especifica qué lados de un elemento deben estar vacíos de otros elementos flotantes.
Ejemplo: clear: both; /* limpia ambos lados */

# flex 
La propiedad flex permite diseñar elementos dentro de un contenedor flex, controlando su distribución y alineación.
Ejemplo: flex: 1; /* crece y se reduce según el espacio disponible */

# grid 
La propiedad grid permite crear un diseño basado en una cuadrícula, organizando elementos en filas y columnas.
Ejemplo: display: grid; grid-template-columns: repeat(3, 1fr); /* 3 columnas de igual tamaño */

# transition 
La propiedad transition permite agregar efectos de transición a cambios en las propiedades CSS.
Ejemplo: transition: all 0.3s ease; /* transición suave en todas las propiedades durante 0.3 segundos */

# transform 
La propiedad transform permite aplicar transformaciones 2D y 3D a un elemento.
Ejemplo: transform: translateX(50px); /* mueve el elemento 50 píxeles a la derecha */

# filter 
La propiedad filter permite aplicar efectos visuales como desenfoque o brillo a un elemento.
Ejemplo: filter: blur(5px); /* aplica un desenfoque de 5 píxeles */

# z-index 
La propiedad z-index controla el apilamiento de los elementos en la página, permitiendo que algunos aparezcan sobre otros.
Ejemplo: z-index: 10; /* elemento se apila sobre otros con menor z-index */

# text-align 
La propiedad text-align alinea el texto dentro de un elemento, afectando su presentación.
Ejemplo: text-align: center; /* alinea el texto al centro del elemento */

# font-size 
La propiedad font-size define el tamaño de la fuente de un texto.
Ejemplo: font-size: 16px; /* establece el tamaño de fuente a 16 píxeles */

# line-height 
La propiedad line-height controla la altura de línea del texto, afectando el espaciado vertical entre líneas.
Ejemplo: line-height: 1.5; /* altura de línea de 1.5 veces el tamaño de la fuente */

# color 
La propiedad color define el color del texto de un elemento.
Ejemplo: color: #333; /* establece el color del texto a un gris oscuro */

# border 
La propiedad border permite establecer el estilo, ancho y color del borde de un elemento.
Ejemplo: border: 1px solid #ccc; /* borde de 1 píxel, sólido, color gris claro */

# text-decoration 
La propiedad text-decoration permite agregar decoraciones al texto, como subrayado o tachado.
Ejemplo: text-decoration: underline; /* subraya el texto */

# cursor 
La propiedad cursor define el tipo de cursor que se muestra al pasar el ratón sobre un elemento.
Ejemplo: cursor: pointer; /* cursor en forma de mano al pasar sobre el elemento */

# visibility 
La propiedad visibility controla la visibilidad de un elemento, permitiendo que ocupe espacio en la página.
Ejemplo: visibility: hidden; /* el elemento es invisible pero ocupa espacio */

# position 
La propiedad position define cómo se posiciona un elemento en la página.
Ejemplo: position: relative; /* posiciona el elemento en relación a su posición original */

# text-transform 
La propiedad text-transform permite modificar la capitalización del texto.
Ejemplo: text-transform: uppercase; /* convierte el texto a mayúsculas */

# box-shadow 
La propiedad box-shadow permite agregar sombras a los elementos, creando un efecto de profundidad.
Ejemplo: box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3); /* sombra a la derecha y abajo con un desenfoque */

# outline 
La propiedad outline define un contorno alrededor de un elemento, similar a un borde pero sin afectar el tamaño del elemento.
Ejemplo: outline: 2px solid blue; /* contorno azul de 2 píxeles */
