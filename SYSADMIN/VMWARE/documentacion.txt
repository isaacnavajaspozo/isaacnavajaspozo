# Me dirijo a la página donde guardo las máquinas
# al crear una máquina nueva se muestra automáticamente la carpeta de referencia por defecto que suele ser: 
C:\Users\<tu-usuario>\Documents\Virtual Machines

# duplico la carpeta perteneciente a la máquina que quiero duplicar

# dentro del duplicado localizo el archivo .vmx en mi caso Windows Server 2019.vmx 
# abro el archivo con un editor de texto y modifico el campo "displayName" por el nombre de mí maquina nueva.

# finalmente, vuelvo a abrir VMWare y ejecuto la nueva máquina.
